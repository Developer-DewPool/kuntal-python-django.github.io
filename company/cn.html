2 Brandcil Portal and Platform Development Python, Django, Django Rest Postgresql

Brandcil is a platform where User can post anything in Facebook, LinkedIn, Twitter at same time by using brancil platform. It is a Multi tenancy Application developed and maintained in python. User can select Package or Subscription plan, one new tenant will be created. Subscriber can Login with that account and create staff and can set permission for those staff. User can upload Image or video or PDF from their Local, Dropbox, Google drive at the time they are sharing the link this will be downloaded and put it into s3 using BOTO3 so that after anytime if user delete those files from their cloud it will still be available into their Brandcil Account. User can select those Image, Video, PDF during posting into social media. People can schedule their post as they want brandcil will post that content in social platform as user mentioned. User can create caption also for future uses. CMS section is also there. Admin can see report and get CSV also. User can upgrade their plan.
