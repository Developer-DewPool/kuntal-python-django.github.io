1 FindMyNearest Python, Django, Scrapy Postgres

This is a new features development project from Realbuzz [realbuzz.com]. We are responsible for scrape the race information from their given listed website after that storing those information into their database. One another section is FMN partner can come to our website and registered their events in our website so that their events reach to the more users. So this app is like a breeze between user as well as company or with event’s organizers. Now when user will login and search a race like 1k, 2k or marathon with a place name, they can see all the available events near to those places.

2 Brandcil Portal and Platform Development Python, Django, Rest API Postgres

For Brandcil [brandcil.com] we are responsible to develop a SAAS base social media management platform where if the User buy a subscription plan for their company or personal purpose they will able to manage all their social media account like Facebook, Twitter, LinkedIn. Subscriber can create user to manage their account on the behalf of that company or individual. Subscriber or user under subscriber can post the same content in Facebook, LinkedIn, Twitter at same time by using Brancil.com platform just with a single click. Post can be schedule as well for posting in a future time. Subscriber/Tenant can upload Image or video from their Local, Dropbox, Google drive, this media objects is being handle with S3 using boto3 package. User can select those Image, Video during posting into social media.
